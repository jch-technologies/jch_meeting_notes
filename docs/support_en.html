Frequently Asked Questions (FAQ)

How do I sync data between devices?

In the app, go to Settings → Data & Sync and choose "iCloud (CloudKit)" for Apple devices or "File Sync" for cross-platform use (e.g. OneDrive, iCloud Drive).

Can I encrypt my sync file?

Yes. When using "File Sync", you can set a password. The file will be encrypted with AES-256. The password is securely stored in the device Keychain.

How do I create a folder?

Open a project and tap the "+" icon next to the project folder header. Folders can be renamed, color-coded, and deleted.

Can I use the app commercially?

Yes. The app may be used for both personal and commercial purposes.

Which platforms are supported?

iPhone, iPad, and Mac (Designed for iPad). Windows support is planned.

My data disappeared after an update?

Check the sync status in Settings. Tap the refresh button on the Home screen to trigger a manual sync. If you use File Sync, make sure the correct sync file is selected.
